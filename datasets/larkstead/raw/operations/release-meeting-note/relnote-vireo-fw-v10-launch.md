Release notes: Vireo firmware v1.0
Released: 2024-10-15
Publisher: Bitgrove Labs (VEN-03) OTA service; posted by Dmitri Okafor
Applies to: all LS-LMP-001 units, first production run

Fixed
- None. Initial release.

Changed
- Initial feature set: Wi-Fi pairing on 2.4 GHz only, up to 6 saved scenes per unit, manual brightness and color-temperature control, and a 5-level sleep-timer dim step-down that eases the lamp off instead of cutting it at the set time.
- OTA update channel established with Bitgrove Labs: connected units check for updates once every 24 hours over the app's background connection, no customer action needed to receive future releases.

Known issues
- Pairing requires a 2.4 GHz network. 5 GHz-only routers are not supported at launch; the setup guide covers enabling a 2.4 GHz band on common router models.

Rollout
Ships pre-installed on every unit from launch, 15 Oct. Pre-order units already boxed before that date receive v1.0 automatically on first connection. Manual path: Vireo app, Settings, Check for update.

Support
Tag pairing tickets vireo-launch for the first 30 days so Dmitri can track volume. The 2.4 GHz limitation is expected behavior, not a defect; point customers to the setup guide before escalating.
