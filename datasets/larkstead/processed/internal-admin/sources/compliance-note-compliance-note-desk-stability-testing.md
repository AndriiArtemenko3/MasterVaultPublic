## Content

Compliance note: Desk line stability testing (tip-test)
Date: 2024-09-18
Author: Dmitri Okafor
Status: informational
Next review: 2025-09-18

Position
3 desk widths on file, 48 in, 60 in, 72 in, all Birch standing desks, all currently hold a passing tip-test result from Timberline Product Testing Lab in Portland. Last full test batch ran 14 Aug 2024: 6 units total, 2 per SKU, pulled off the Ostrava Metalworks (VEN-02) production run that started 02 Jul 2024 carrying the revised leg bracket casting. No tip recorded on any of the 6 units at 65 lb applied at the leading edge, full extension, both static and dynamic passes. The prior batch, tested 11 Mar 2024 against the old bracket casting, also passed clean, so the bracket change did not need to happen for safety reasons, it happened for a cost reduction on the Ostrava side and we retested anyway because the geometry moved.

Obligations
| obligation | detail | where handled |
|---|---|---|
| tip-test pass, all desk SKUs | static and dynamic tip load, 65 lb at the leading edge, full extension, no tip on any tested unit | Timberline Product Testing Lab, report on file |
| retest trigger | any change to leg, bracket, or crossbar geometry from Ostrava Metalworks triggers a full retest before the changed units ship to customers | Operations, Dmitri Okafor |
| file retention | test reports and raw load data kept 5 years from test date | Operations shared drive |

Open items
- If Ostrava proposes another bracket or crossbar revision, then schedule the retest before those units clear receiving, not after. Owner DO, no date set, no revision pending as of this note.
- LS-DSK-001-72 only had 2 units in the last batch, same as the 48 in and 60 in runs. Worth pulling a 3rd unit on the 72 in next cycle given it carries the most load at full extension. Owner DO, decide by next review 2025-09-18.
