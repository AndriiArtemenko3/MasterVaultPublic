## Content

To: support team, Mara Voss
From: Ana Petrova
Date: 2025-09-08
Subject: [APPROVAL] Refund exception, LOT-2025-14 Alder mats

Formalizing what support has already been doing since the macro went
live. Any Alder mat that traces to LOT-2025-14 is refundable or
replaceable regardless of when it was purchased, through 2025-12-31, no
restocking fee. This supersedes the standing 30-day refund window
(effective 2024-01-15) for this lot only, nothing else changes on that
policy.

If the mat is LOT-2025-14, the 10% restocking fee never applies, because
these are defective units, not opened non-defective returns, and the fee
was never meant to cover that case.

This exception runs retroactively from 2025-08-06, the day the support
macro went live, so it covers everything handled under the macro from
that date forward, not just claims filed after today. We don't want a
customer whose ticket happened to land on 2025-08-07 treated differently
from one who wrote in this week.

Cumulative program cost against this incident is 2056.35 to date, which
crosses the 1000.00 CEO approval threshold in the expense policy
effective 2025-07-01. That figure is refunds, replacement stock, outbound
shipping, and the scrap write-off combined, before the Verdant credit is
applied against any of it. Mara Voss approved this exception and the
cumulative cost on 2025-09-08.

If a mat comes in after 2025-12-31 still tagged to this lot, that's a
judgment call for support, not an automatic exception under this memo.
I'd rather revisit the cutoff in December with real numbers than guess
now at how many stragglers there will be.

Priya: make sure the exception language matches what's already in the
macro so support isn't running two slightly different versions of the
same thing.

Ana
