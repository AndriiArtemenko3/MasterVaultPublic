Position
Larkstead Goods Company, LLC filed its 2025 annual report with the state's corporate registry office on 2025-05-29, ahead of the 01 Jun deadline that runs off the 2022-06-01 formation date. Filing fee 100.00, paid by company card, self-approved under the 250.00 threshold in force as of this date, no manager sign-off needed. Willamette Registered Agent Services holds the registered agent role at their Salem, Oregon address, the same agent on file since formation, and that contract renews each March, current term through 2026-03-01. If the registered agent lapses, the annual report filing can be rejected outright, so the two obligations sit next to each other in this note on purpose rather than as separate items someone might forget to cross-reference later.

| filing fee | 100.00, self-approved, paid by company card | Ledgerly, expense code ADMIN-LEGAL |
