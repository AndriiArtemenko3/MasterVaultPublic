# Vireo Goodwill Warranty Memo

## Summary

To: all staff From: Mara Voss Date: 2026-01-20 Subject: Vireo warranty extension, effective today Warranty on affected Vireo units goes to 2 years, effective today. Goodwill only, tied to the firmware issue.

## Content

To: all staff
From: Mara Voss
Date: 2026-01-20
Subject: Vireo warranty extension, effective today

Warranty on affected Vireo units goes to 2 years, effective today. Goodwill only, tied to the firmware issue. Not a change to standard policy.

Scope is vireo-v12-affected: 618 LS-LMP-001 units on Meridian PCB rev C that took firmware v1.2. Every other Vireo unit stays on the 1 year standard warranty from 2024-01-15, rev C included if it never took v1.2. Check the serial against the affected list first. Unsure, treat it as standard and escalate to me.

Cobalt Dental Group's two add-on units are in scope. Those are the ones on order #LS31695, account PW-cobalt-dental-40seat, delivered with wave 1 on 2025-12-10. Tom, make sure that's reflected if this comes up in the renewal conversation.

Priya: update the public Vireo help doc to reflect the 2-year extension for affected units. This is customer-facing and needs to be accurate.

Finance should track this scope in Helprise going forward so we can quote it consistently on any future ticket, not just the ones already open. [[support]] agents should update the ticket tags on all 12 open flicker tickets to reflect the new coverage today.

We still show 26 units offline since the rollback closed on the 13th. Who owns getting those back online, and by when?

M
