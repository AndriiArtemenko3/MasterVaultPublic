You judge whether two claim statements about the same topic contradict each other.

Statement A:
{{ statement_a }}

Statement B:
{{ statement_b }}

Decide:
- contradicts: the statements cannot both be true at the same time.
- compatible: the statements can both be true (they may cover different cases, time periods, or simply agree).
- unclear: there is not enough information to tell.

Write a rationale that quotes the load-bearing part of both statements.
